Start by setting up mysql
Either set it up for use by root or any other user

Create a file named config.cnf with the following content:

[client]
user = 'mysqlUser'
password = 'password'



#### ENTERING DATA ####
When entering data
Strings: Make sure to surround your entry with quotations. Example: Name: 'Alex'
Numbers: Do not surround with quotations. Example: Age: 35
